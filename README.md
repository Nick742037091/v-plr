## Install

`$ npm install v-plr or yarn add v-plr`

## Quick Start

```
import VideoPlayer from 'v-plr'
Vue.use(VideoPlayer)


<template>
  <video-player src="http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4" :title="video"/>
 </template>
```
